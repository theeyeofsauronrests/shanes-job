# Shane's Job List Documentation

Docs for Shane Quinlan's unofficial Product, Design, and Engineering jobs list for defense technology companies.

## Files

- `PRD.md` - product scope, data model, acceptance criteria
- `DESIGN.md` - DefSimple-inspired visual direction
- `SCRAPING.md` - public board scraper behavior and safety rules
- `API.md` - static JSON contract
- `DEPLOYMENT.md` - static hosting notes
- `WORKFLOW.md` - development workflow manifest
- `workflows/feature-development.md` - issue-to-merge workflow
- `../data/README.md` - manual CSV import guide

## Architecture

- Vite
- React
- TypeScript
- Plain CSS
- Vitest
- Playwright
- Static JSON data

No backend, login, database, secrets, analytics, telemetry, or applicant data collection.
